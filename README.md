# Random Word Generator 🎲

* Random Word Generator is a lightweight and interactive web-based tool that generates random words instantly with a clean and modern interface.

This project provides a simple way to generate creative ideas, project names, writing inspiration, or random word combinations. Built with pure **HTML, CSS, and JavaScript**, it runs entirely in the browser and can be deployed easily using **GitHub Pages**.

---

# 🎲 How to Use

1. Open the Random Word Generator website.
2. Click the **Generate** button.
3. A random word will appear instantly.
4. Use the **Copy** button to copy the generated word to your clipboard.
5. Use the generated words for:

   * 💡 Creative ideas
   * 📝 Writing inspiration
   * 💻 Project naming
   * 🎨 Brainstorming
